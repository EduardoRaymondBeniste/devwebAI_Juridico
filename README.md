# DevWebAI - Engenharia de Software Jurídica ⚖️

Site oficial da **DevWebAI**, focado em soluções tecnológicas para o setor jurídico.

## 🚀 Tecnologias
- **Front-end:** HTML5, CSS3 (Bootstrap 5), JavaScript
- **Back-end:** Node.js & Express
- **Banco de Dados:** PostgreSQL (Neon) / MySQL (Local)

## 🛠️ O Projeto
Desenvolvimento de arquitetura de software para alta performance operacional em escritórios de advocacia.
